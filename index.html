About This Site

This is a simple authentication demo that shows:

  • User registration
  • Admin registration via QR code
  • User login
  • Protected content that's only visible to logged-in users
  • Different access levels for admins and regular users

If you try to access the protected page without being logged in, you'll be redirected to the login page.

Admin Registration

Administrators can register by scanning a QR code that comes with the SafeKey box. The QR code contains a link to a special registration page with the box's serial number. Try the admin registration with a demo serial number.
