Willkommen bei Ihrer SaveKey Schlüsselbox Einrichtung

Auf dieser Seite finden Sie eine einfach zu befolgende Anleitung, die Sie durch den gesamten Prozess der Einrichtung Ihrer SaveKey Schlüsselbox führt. Wir haben die Anleitung in klare Schritte unterteilt, damit Sie ohne Probleme starten können.

Folgen Sie einfach den Schritten nacheinander. Bei Fragen steht Ihnen unser Support-Team jederzeit zur Verfügung.

Übersicht der Einrichtungsschritte

  • 1 QR-Code scannen und Admin-Konto erstellen
  • 2 Hardware einrichten
  • 3 Erste Anmeldung als Administrator
  • 4 RFID/NFC-Chip zuweisen
  • 5 Weitere Benutzer informieren
  • 6 System konfigurieren
  • 7 System testen
1

QR-Code scannen und Admin-Konto erstellen

Der erste Schritt ist das Einrichten Ihres Administrator-Kontos:

  • Scannen Sie den QR-Code, der mit Ihrer Schlüsselbox geliefert wurde. Dieser Code enthält die einmalige Seriennummer Ihrer Box.
  • Alternativ können Sie den beiliegenden Link in Ihrem Webbrowser öffnen.
  • Führen Sie die Registrierung durch, indem Sie alle erforderlichen Daten eingeben.
  • Wichtig: Bewahren Sie Ihre Zugangsdaten sicher auf. Sie werden sie benötigen, um als Administrator auf das System zuzugreifen.
2

Hardware einrichten

Sobald Sie Ihr Admin-Konto erstellt haben, können Sie die Schlüsselbox einrichten:

  • Verbinden Sie die Schlüsselbox mit einer Stromquelle.
  • Stellen Sie sicher, dass WLAN in der Nähe verfügbar ist.
3

Erste Anmeldung als Administrator

Jetzt können Sie sich zum ersten Mal anmelden und Ihr System konfigurieren:

  • Öffnen Sie die SaveKey-Webseite und klicken Sie auf "Login".
  • Melden Sie sich mit Ihren bei der Registrierung erstellten Admin-Zugangsdaten an.
  • Nach der Anmeldung werden Sie zum Admin-Dashboard weitergeleitet.
4

RFID/NFC-Chip zuweisen

Als Administrator können Sie nun Ihren RFID/NFC-Chip für den Zugriff auf die Schlüsselbox einrichten:

  • Gehen Sie im Admin-Dashboard zum Menüpunkt "Mein Profil".
  • Wählen Sie die Option "RFID/NFC-Chip zuweisen".
  • Halten Sie den Chip an das Lesegerät der Box, wenn Sie dazu aufgefordert werden.
  • Bestätigen Sie die Zuweisung im System.
5

Weitere Benutzer informieren

Weitere Benutzer müssen sich selbst registrieren, um Zugriff auf die Schlüsselbox zu erhalten:

  • Teilen Sie die Seriennummer der Schlüsselbox mit den Personen, die Zugriff haben sollen.
  • Informieren Sie die Benutzer, dass sie sich auf der SaveKey-Website unter "Registrieren" anmelden müssen.
  • Wichtig: Jeder Benutzer muss bei der Registrierung die korrekte Seriennummer der Schlüsselbox eingeben.
  • Nach erfolgreicher Registrierung können die Benutzer ihren eigenen RFID/NFC-Chip zuweisen.
6

System konfigurieren

Passen Sie die Einstellungen an Ihre Bedürfnisse an:

  • Gehen Sie im Admin-Dashboard zum Menüpunkt "Einstellungen".
  • Konfigurieren Sie Push-Benachrichtigungen für Ereignisse wie Schlüsselentnahme oder Warnungen.
  • Legen Sie Zeitfenster fest, in denen bestimmte Benutzer Zugriff haben sollen (optional).
  • Speichern Sie Ihre Einstellungen.

Tipp:

Es empfiehlt sich, die Push-Benachrichtigungen zu aktivieren, um über wichtige Ereignisse direkt informiert zu werden.

7

System testen

Zum Abschluss sollten Sie das System testen:

  • Testen Sie die Entnahme und Rückgabe des Schlüssels mit Ihrem registrierten RFID/NFC-Chip.
  • Überprüfen Sie, ob die konfigurierten Push-Benachrichtigungen korrekt auf Ihrem Smartphone ankommen.
  • Prüfen Sie, ob der Zugriff nur mit registriertem RFID/NFC-Chip möglich ist.
  • Überprüfen Sie, ob alle Aktivitäten korrekt im Protokoll angezeigt werden.
